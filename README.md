# Vinyasa
---
  An E-Commerce website for brand new clothes
</div>

### Built with:

- ReactJS
- React router
- Prime React UI
- CSS
- Axios
- Mockbee(backend)

### How to install and run locally ?

```
$ git clone https://github.com/varunjram/vinyasa.git
$ cd vinyasa
$ npm install
$ npm start
```

## Features -

- Home Page
- Product Listing Page
- Single Product Page
- Filters by
  - Price
  - Category
  - Ratings
- Sorting by price
- Wishlist Management
- Cart Management
- Search
- Loading & Alerts
- Authentication
- Sign-up
- Login
- Logout
- Address Management
- Checkout
- Order Summary
